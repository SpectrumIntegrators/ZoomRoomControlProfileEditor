Privacy Statement

Last updated: 2026-05-14

Summary

The Zoom Rooms Control Profile Builder runs entirely in your browser. There is no backend operated by Spectrum Integrators that collects, stores, or processes the profiles you build, edit, or download. Profile data stays on your device unless you deliberately share or download it.

What stays on your device

  • The profile JSON you are editing — never leaves your browser unless you click Download.
  • Layout preferences (splitter pane sizes) — stored in your browser's localStorage so the editor remembers them between visits. Clearing your site data removes them.

What we do not collect

  • No analytics, telemetry, or session recording.
  • No cookies.
  • No account or sign-in.
  • No third-party advertising or tracking scripts.

Third-party services this page communicates with

Loading the editor causes your browser to fetch a small number of static assets from third-party hosts. None of these services receive your profile data — they only see the standard HTTP request for the static asset (URL, IP address, user-agent, timestamp).

  • GitHub Pages — hosts the editor itself. GitHub records standard web request logs for security and reliability. See the GitHub General Privacy Statement.
  • jsDelivr — serves the JSON profile schema. See the jsDelivr privacy policy.
  • Google Fonts — serves the Material Icons glyphs and the Lato webfont used by the UI. See the Google Privacy Policy.

Questions

For questions about this statement or how the editor handles data, contact Spectrum Integrators or open an issue on the project repository.
